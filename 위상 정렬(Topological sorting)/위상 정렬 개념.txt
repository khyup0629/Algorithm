## 위상 정렬

+ **사이클이 없는 방향 그래프**의 **모든 노드를 방향성에 거스르지 않도록 순서대로 나열**하는 것을 의미한다.
+ 예시) 선수과목을 고려한 학습 순서 설정

+ 위 세 과목을 모두 듣기 위한 적절한 학습 순서는?
	+ 자료구조 -> 알고리즘 -> 고급 알고리즘 (O)
	+ 자료구조 -> 고급 알고리즘 -> 알고리즘 (X)

### 진입차수와 진출차수

+ 진입차수(Indegree) : 특정한 노드로 들어오는 간선의 개수
+ 진출차수(Outdegree) : 특정한 노드에서 나가는 간선의 개수
