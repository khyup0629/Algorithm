## 별표(★)의 의미

+ 없음 : 직접 풀었으며 **개선할 필요 없는** 코드
+ (★) : 직접 풀었으나 **개선이 필요**했던 코드
+ (★★) : 풀지 못했고, **아이디어를 참고**해야 했던 코드
+ (★★★) : 풀지 못했고, 아이디어를 참고해도 **이해가 되지 않는** 코드
